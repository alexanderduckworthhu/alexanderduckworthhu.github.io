Humanitarian data · Geospatial analytics

Where Needs Overlap

Code ↗

A country-level dashboard showing where acute food insecurity, hosted displacement, and under-5 mortality coincide — built for IM and programme analysts working with UNICEF, UNHCR, and OCHA counterparts.

Challenge & trade-offs

  • Joined three live public APIs (HDX/IPC, UNHCR, WHO GHO) on ISO3 with documented definitions — origin vs asylum counts answer different questions, so country-of-asylum was locked in.
  • Deliberately no ML forecasting: unverifiable crisis prediction would undermine trust; the tool frames one decision question instead.

Impact

  • Surfaced 3 dual-pressure countries (Sudan, Ethiopia, Chad) above both focus-set medians across ~48.6M people in IPC Phase 3+.
  • Coverage gaps flagged explicitly rather than silently imputed.

NLP · Retrieval-augmented generation

WHO Guidance QA — Multilingual RAG

Code ↗

A document QA assistant over WHO topic guidance that indexes English and French in one vector space, answers with passage-level citations, and refuses when retrieval is weak.

Challenge & trade-offs

  • Chose multilingual-e5-small over OpenAI embeddings for a shared EN/FR space and fully offline demos; Chroma over Pinecone for zero cloud cost and inspectable indexes.
  • Trust stack: distance gate → closed-context prompt → chunk-ID citations → verifiable source passages in the UI. No "zero hallucination" claims — retrieval quality is the measured engineering claim.

Impact

  • Precision@5 = 0.65 · MRR = 0.938 on a golden-query eval set, reproducible with one command.
  • Works without any API key via an extractive fallback path.

Clinical ML · Explainability

ICU 48-hour Mortality Risk with SHAP Attribution

Code ↗

Calibrated short-horizon mortality risk from vital-sign time series — plus a SHAP heatmap showing which vital × lookback window pushed each score, so clinicians can agree, challenge, or investigate.

Challenge & trade-offs

  • Isotonic-calibrated logistic regression before any deep model: most MIMIC tutorials stop at AUROC and skip calibration, missingness honesty, and how the number should be spoken aloud.
  • Transparent preprocessing (plausibility gates, missingness flags, bounded forward-fill) chosen over MICE because opaque imputation is harder to defend in clinical review.

Impact

  • Full pipeline from raw vitals to calibrated probability + attribution, reproducible in three commands; MIMIC-III integration path documented under PhysioNet DUA.
  • Dedicated ethics & uncertainty guidance shipped alongside the model.

Composite indices · Climate risk

Climate Migration Risk Index

Code ↗

A weight-defendable, scenario-stressable country index of climate-linked mobility pressure — combining climate, socioeconomic, conflict, and food fragility pillars on an interactive choropleth.

Challenge & trade-offs

  • Transparent fixed pillar weights over PCA/AHP so an analyst can challenge every prior; a sensitivity tab compares equal-weight rankings.
  • 1.5 °C / 2 °C scenarios scale the climate pillar only — communicating scenario thinking honestly without claiming a full climate model.

Impact

  • 42 countries scored; rank validation against ND-GAIN-like (ρ = 0.86) and INFORM-like (ρ = 0.90) references.
  • Every build writes provenance metadata; weights are versioned in config.

ESG Composite Scoring

Code ↗

An auditable E/S/G composite for 25 public issuers that answers the client question vendor letter-grades can't: "why does this name look strong or soft?"

Challenge & trade-offs

Predictive maintenance · IoT / OT

CNC Spindle Health Monitor

Code ↗

A Plotly Dash floor board that turns multivariate sensor streams into per-cell health scores, time-to-failure estimates, and maintenance notes — designed for precision-manufacturing environments where failure labels don't exist on day one.

Challenge & trade-offs

  • Unsupervised Isolation Forest on windowed statistics instead of an LSTM autoencoder — works without labeled failures and stays explainable to plant engineers.
  • Mock OPC-UA tag layer (ns=2;s=CNC-01.Spindle…) demonstrates real IT/OT architecture without pretending plant access; NASA CMAPSS degradation structure remapped to CNC vocabulary with the mapping documented.

Impact

  • 182 scored windows across a 4-machine fleet with traffic-light thresholds matching shop-floor language (green ≥ 75 / amber ≥ 50 / red).
  • One click on a cell card refreshes sensor traces and maintenance notes; 8-language UI.

Wild Log

Code ↗

An offline personal expedition log for outdoor athletes — session history, map memory, and honest consistency metrics across 7 activity types, with no account, no cloud, and no social feed.

Challenge & trade-offs

Research paper

First-author work on applying self-supervised learning to conservation-scale imagery.

Computer vision · Self-supervised learning · Edge AI

Self-Supervised Anomaly Detection for Wildlife Conservation and Biodiversity Monitoring

Read PDF ↗ Code ↗

Alexander Duckworth-Hu & Parkpoom Chaisiriprasert · Department of Digital Innovation Technologies, Rangsit University (2026)

A system-design proposal for detecting conservation-critical events — poaching, injured animals, invasive species — by learning what "normal" looks like from unlabeled camera-trap imagery, so the model flags anything statistically novel without a single manual label.

The problem & the shift

  • Camera traps, drones, and satellites generate petabytes of imagery; the rarest, most safety-critical events (a poacher, a new invasive species) are exactly the ones supervised classifiers never saw in training and silently misclassify.
  • Reframes the question from "which known class is this?" to "does this deviate from the learned normal distribution?" — an open-set framing far better matched to conservation's open-ended threats.

Approach

  • SimCLR contrastive learning + Masked Autoencoding on a Vision Transformer, with temporal pretext tasks over camera-trap bursts; per-pixel reconstruction error becomes the anomaly score.
  • Multi-modal fusion of camera-trap RGB, drone thermal, and satellite multispectral streams, optimized for edge inference on NVIDIA Jetson with TensorRT FP16.

Evidence & projections

  • Evaluated against 3 public LILA BC benchmarks totaling 8.74M images (Snapshot Serengeti, WCS/iWildCam, Caltech Camera Traps).
  • A proof-of-concept on 1,150 Serengeti frames beats a pixel-MSE baseline (AUC 0.561 vs 0.544) with statistically significant error separation (p = 0.020) — confirming 700 unlabeled images suffice to learn a normality representation.
  • Theoretical projections: 0.93 F1 for poacher detection, 0.86 for injured animals, with ~0.04 cross-ecosystem transfer penalty.

Deployment realism

  • Complete bill of materials at $1,718 per solar-powered station (45 FPS at 10 W on Jetson Xavier NX) and a six-step field protocol requiring no ML expertise from rangers.
  • Honest scoping throughout: headline accuracy figures are labeled theoretical projections, with live-deployment validation named as the primary future work.
